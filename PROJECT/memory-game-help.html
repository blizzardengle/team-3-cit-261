Purpose of the Game

This game is design to help you exercise your memory as you play to find the match between the therms and definitios found on the flash cards you have already created.

How to play the game

  1. Click on the game menu
  2. Choose the memory game option
  3. Start Playing by clicking on one card
  4. Find the match for the card
  5. Once you have found each term with its respective definition then you have won

To play again just click on the start button
